Please configure your phones to forward emergency calls to the PSTN gateway directly. You should use the caller sensitive forwarding feature only if your phone does not support direct forwarding to a gateway for emergency calls.

If this feature is enabled all emergency calls will be forwarded to the default gateway. Optionally you can configure a set of phone numbers that will be forwarded to the alternative gateway.
